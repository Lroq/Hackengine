🛠️ Outils d'Édition

🖌️ Pinceau
Place des tiles une par une
• Clic gauche maintenu : dessine en continu
• Idéal pour les détails précis
🪣 Pot de Peinture
Remplit une zone adjacente avec la tile sélectionnée
• Clic gauche sur une zone : remplit automatiquement
• Parfait pour remplir de grandes surfaces rapidement
🧹 Gomme
Efface des tiles existantes
• Clic gauche maintenu : efface en continu
• Supprime toutes les tiles sur la position (tous layers)

⌨️ Raccourcis Clavier

Annuler la dernière action Ctrl + Z
Refaire une action annulée Ctrl + Y
Sauvegarder la map manuellement Ctrl + S
Fermer les menus et modales Échap

💡 Historique de 50 actions maximum

🖱️ Contrôles Souris

🎨 Panneau des Assets

• Créez des dossiers pour organiser vos assets

💡 Structure sauvegardée automatiquement
🔵 Sélection d'Asset
L'asset sélectionné a un contour bleu
• Cliquez sur une tile pour la sélectionner
📤 Upload de Tiles
Cliquez sur "➕ Tuile" pour uploader une image PNG
• Format recommandé : 27×27 pixels
• Formats supportés : PNG, JPG, GIF, SVG

⚙️ Configuration des Tiles

🎨 Layers (Plans)
Configuré AVANT placement dans le panneau Assets :
• Layer 0 (Sol) : Arrière-plan, herbe, routes...
• Layer 1 (Murs/Déco) : Murs, objets, décorations...
• Layer 2 (Sprites) : Personnages, éléments au premier plan
💡 Les layers s'empilent : 0 derrière, 2 devant
🛡️ Collision (Solide/Non Solide)
Configuré AVANT placement dans le panneau Assets :
• Non Solide : Le joueur peut traverser (herbe, sol...)
• Solide : Bloque le joueur (murs, obstacles...)
🧱 Indicateur rouge sur les tiles solides en mode construction
🌀 Téléporteurs
Configuré APRÈS placement via Alt+Clic :
1. Placez une tile
2. Alt+Clic sur la tile
3. Activez "🌀 Téléporteur"
4. Configurez : Map de destination + Coordonnées (en tiles)
💡 Utilisez "📍 Copier position actuelle du sprite" pour les coordonnées
💡 1 tile = 27 pixels (ex: X=5 → position 135)
🌀 Indicateur bleu sur les téléporteurs en mode construction

🧭 Navigation & Caméra

🎯 Centrer sur Sprite
Bouton dans le panneau de gauche
• Téléporte instantanément la caméra sur votre personnage
• Utile après avoir exploré loin sur la carte
⊕ Aller au Centre (0;0)
Bouton dans le panneau de gauche
• Téléporte la caméra au centre exact de la grille
• Point de référence marqué par une croix rouge/verte
🖱️ Déplacement Manuel
Clic droit maintenu + mouvement de la souris
• Navigation libre dans toute la zone de construction

🗺️ Gestion des Maps

➕ Créer une Nouvelle Map
Bouton "🗺️ Maps" → "➕ Nouvelle Map"
• Choisissez un nom unique
• Sélectionnez la taille : Petite (25×25), Moyenne (50×50), Grande (75×75), Très Grande (100×100) ou Personnalisée
🔄 Changer de Map
Bouton "🗺️ Maps" → Sélectionnez une map
• Les anciennes tiles sont automatiquement nettoyées
• La grille s'adapte à la taille de la nouvelle map
🗑️ Supprimer une Map
Bouton "🗑️" sur la carte dans le sélecteur
⚠️ Action IRRÉVERSIBLE - Toutes les tiles seront perdues

💾 Sauvegarde

⏰ Sauvegarde Automatique
Toutes les 60 secondes automatiquement
• Aucune action nécessaire
✓ Notification "🕒 Sauvegarde automatique..." dans la console
💾 Sauvegarde Manuelle
Bouton "💾 Sauvegarder" ou Ctrl+S
• Sauvegarde immédiate de la map actuelle
✓ Feedback visuel : bouton devient vert "✅ Sauvegardé !"

📏 Limites & Règles

  • ✓ La grille de placement est configurable par map (25×25 à 200×200)
  • ✓ Le centre (0;0) est marqué par une croix rouge (horizontal) et verte (vertical)
  • ✓ Les tiles ne peuvent être placées que dans la bordure blanche de la grille
  • ✓ La grille visuelle s'adapte automatiquement à la taille de la map
  • ✓ Jusqu'à 3 layers peuvent coexister sur une même position
  • ✓ Historique de 50 actions (Ctrl+Z / Ctrl+Y)
  • ✓ Sauvegarde automatique toutes les 60 secondes

👁️ Indicateurs Visuels

🎨 Modes
• 🧱 Mode Construction - Bordure bleue dans le header
• ▶️ Mode Play - Bordure verte dans le header
🏷️ Propriétés des Tiles (mode construction uniquement)
• 🧱 Icône rouge = Tile solide (collision activée)
• 🌀 Icône bleue = Téléporteur configuré
• Les deux peuvent coexister sur une même tile
📍 Grille & Repères
• Grille blanche : zone de placement des tiles
• Croix rouge (horizontale) + verte (verticale) = centre (0;0)
• Bordure blanche : limite de la grille
🔵 Sélections
• Contour bleu = Asset sélectionné dans le panneau
• Fond bleu actif = Outil d'édition actif (Pinceau/Pot/Gomme)

💡 Astuces & Tips

🎨 Organisation
Créez des dossiers thématiques : "Sol", "Murs", "Décorations", "Personnages" pour retrouver vos assets facilement
⚡ Workflow Rapide
1. Sélectionnez le Layer + Collision AVANT de placer → 2. Utilisez le Pot de peinture pour les grandes zones → 3. Peaufinez avec le Pinceau
🌀 Téléporteurs
Positionnez votre sprite là où vous voulez arriver, puis copiez les coordonnées avec le bouton 📍 - Plus besoin de calculer !
💾 Sauvegarde
Pas de stress : La sauvegarde automatique fonctionne toutes les minutes. Utilisez Ctrl+S pour être sûr après un gros changement
🔄 Erreurs
Vous avez fait une erreur ? Ctrl+Z annule jusqu'à 50 actions. Vous pouvez même Ctrl+Y pour refaire si vous annulez trop !
📏 Layers Multiples
Utilisez Layer 0 pour le sol, Layer 1 pour les murs/objets, et Layer 2 pour les éléments décoratifs au premier plan. Ça crée de la profondeur !
